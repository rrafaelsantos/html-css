Capítulo 1

Capítulo 1.1

Neste capítulo aprenderemos um pouco mais sobre a inserção de estilos em CSS no modo de Externos.

Este tipo de aplicação de estilos é o mais utilzado no mundo dos programadores. visto que ele é o mais geral possível, visto que é aplicável em toda página que o desenvolvedor quiser.

Capítulo 1.2

Agora, parecido com o modo local, usaremos incluíremos uma tag dentro do bloco <head>. Entretando, agora utilizaremos a tag link
Por Exemplo:

<link rel="stylesheet" href="style.css">

Onde em "href" é o nome do arquivo CSS3

Capítulo 2

Capítulo 2.1

Este estilo é utilizado na maioria dos sites, pois a esmagadora maioria das empresas precisam de diversas páginas para a melhor navegação do cliente. Além de que este tipo de arquivo é facilmente linkável, sem precisar ficar incluindo o código do estilo inteiro a cada página que o desenvolvedor precisa criar, diminuindo assim, tempo, e, principalmente, a quantidade de códigos digitados nos leitores.
