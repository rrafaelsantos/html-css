Principais formatações

Negrito / Destaque

Nessa frase temos um termo em negrito usando a tag <b> (não semântica)

Nessa frase, tenho um termo em destaque usando a tag <strong> (semântica)

Itálico / Ênfase

Nesta frase, temos um termo itálico usando a tag <i> (não semântica)

Nesta frase, temos um termo em ênfase usando a tag <em> (semântica)

Texto marcado

Podemos criar também um texto marcado usando a tag <mark>

A cor original é amarelo marca-texto, porém é possível mudar no CSS

Texto deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido mas desconsiderado com a tag <del>

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi indicado depois com <ins>

Existe também o sublinhado com a tag <u> (não semântica)

Texto sobrescrito

<sup> Para inserir coisas do tipo x20+3

Texto subscrito

<sub> Para inserir coisas do tipo H20

OBS: Nesta aula também aprendemos a envelopar. Selecione o que deseja envelopar, clique CTRL + SHIFT + p, depois selecione a opção >Emmet: Wrap with abbreviation e, por fim, a tag que deseja aplicar
