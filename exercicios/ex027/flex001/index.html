Flexible Box

Os Flexible boxes são uma forma de responsividade nos site.

Ele deve ser utilizado quando o conteúdo precisa se enquadrar ao layout do container e ao tamanho da tela do usuário.

Flexible Box Module

O container (pai) levará a seguinte configuração: "display: flex;"

Já os itens (filhos) receberão outros tipos de configuações.
