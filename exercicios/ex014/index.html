Estilos Locais

Os estilos Locais (ou internos) são aplicados de maneira mais geral comparado ao inline.

Para utilizá-lo é necessário colocar uma tag <style> dentro do bloco <head>.

Um exemplo abaixo do estilo que estamos usando nesse site.

    </title>
        <style>
            body {
                background-color: lightsteelblue;
                font-family: Arial, Helvetica, sans-serif;
                font-size: 20px;
            } 
            
            h1 {
                color: darkblue;
            }
            
            h2 {
                color: darkred;
            }
            
            p {
                color: darkmagenta;
            }
        </style>
    </head>
